# 產品藍圖：AI工作室

## 願景

讓 AI 工作室能持續自我評估、自動產出並安全落地改良，越用越強

## 目標用戶

用 Ti Studio 開發產品的團隊與維運者，需要工作室在交付之餘也能自我精進

## 核心功能

- **[P0]** 能力自評記分卡 — 每場 session 結束量化成果（測試/Demo/審查通過率）並跨場聚合趨勢
- **[P0]** 自我改良 backlog — 從失敗與檢討自動萃取「找問題→列待辦」並排優先序
- **[P0]** 安全自改閉環 — 核心改動走獨立 PR＋測試 gate，綠燈才合併部署，絕不污染專案 repo
- **[P1]** 改良成效驗證 — 每次自改前後跑同組基準，量化是否真的變強、可回滾
- **[P1]** 額度感知分派 — 依 provider 即時額度自動選最寬鬆後端，避免卡死
- **[P1]** 動態招募專家 — PM 依任務缺口臨時擴編角色/小組，提升難題命中率
- **[P2]** 改良歷程儀表板 — 視覺化自我進步軌跡與每次改動的影響面
- **[P2]** 經驗教訓沉澱 — 把踩坑自動回寫協作記憶，避免重犯

## 里程碑

- M1 能力自評記分卡＋自我改良 backlog＋安全自改閉環（可量化現況、產待辦、安全落地一次改良）
- M2 改良成效驗證＋額度感知分派＋動態招募（自改可證明變強、跑得久、攻得下難題）
